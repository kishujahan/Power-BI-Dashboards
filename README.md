# Data Jobs Dashboard with Power BI

## 📌 Introduction

This project presents an interactive Power BI dashboard analyzing data **job trends, salaries, and demand** using a real-world dataset of 2024 data science job postings. It explores key metrics such as job count, salary distribution, job types, and global demand to provide insights into the data job market.

The dashboard was created as part of the Power BI learning process by following the tutorial by [Luke Barousse](https://youtu.be/FwjaHCVNBWA?si=CW3iSXzXALXmF9UR). The project focuses on **practicing data visualization, dashboard design, and insight-driven analysis using Power BI**, without modifying the original guided workflow.

## 🧠 Skills Showcased


- **Data Visualization:** Created column, bar, and line charts to effectively present trends, comparisons, and patterns in job data.  
- **Geographic Analysis:** Used map visuals to analyze the global distribution of job opportunities across different regions.  
- **KPI Metrics:** Designed cards to highlight key metrics such as job count and average salary for quick insights.  
- **Tabular Analysis:** Used a matrix visual to display yearly and hourly salaries, job count, and trends, enhanced with sparklines for quick comparison over time. 
- **Dashboard Design:** Built a clean and structured dashboard layout for easy navigation and better user experience.  
- **Interactivity:** Implemented slicers to allow users to filter data dynamically based on job Title.  
- **Navigation:** Used buttons and bookmarks to enable smooth navigation between different views within the dashboard.  
- **Advanced Features:** Applied drill-through functionality to navigate from overview data to more detailed insights.

## 📊 Dashboard Overview

### Page 1 : Job Market Overview

![Dashboard Page 1 ](/images/proj1_page1.PNG)

This page provides a high-level view of the data job market. It includes job count and salary benchmarks across various roles, allowing for a direct comparison between yearly and hourly compensation. This view helps identify which positions are dominating the market and how pay scales correlate with specific job titles.

### Page 2 : Job Role Deep Dive

![Dashboard Page 2 ](/images/proj1_page2.PNG)

This drill-through page offers a detailed profile of a specific job title, focusing on the finer details. It evaluates key factors such as remote work availability, educational requirements, and standard benefits like health insurance. Furthermore, it identifies the primary recruitment platforms and geographic hotspots for the selected role, providing a comprehensive breakdown of the typical work environment and contract types.
